Materials for the paper **Progressive Cramming: Reliable Token Compression and
What It Reveals** (Tarasov, Lashukov, Goncharova, Kuznetsov).

presentation lives in [`presentation/`](./presentation) and is deployed to GitHub
Pages automatically on every change (see
[`.github/workflows/deploy-presentation.yml`](./.github/workflows/deploy-presentation.yml)):
